# Nexus Gaming Launcher

Modern, lightweight and customizable game launcher built with Python.

## About

Nexus Gaming Launcher is a simple and fast launcher that allows you to organize and run your games from one central interface. It focuses on clean design, ease of use, and performance.

## Features

- Clean and modern UI  
- Fast game launching  
- Easy game library management  
- Lightweight Python application  
- Customizable layout and design  

## Installation

```bash
git clone https://github.com/your-username/nexus-gaming-launcher.git
cd nexus-gaming-launcher
pip install -r requirements.txt
python main.py
```

## Requirements

- Python 3.8+
- Required libraries listed in requirements.txt

## Usage

- Add your games to the launcher  
- Select a game and launch it instantly  
- Customize the interface as you want
